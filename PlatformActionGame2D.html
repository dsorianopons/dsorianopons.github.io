Platform Action Game 2D

 

This project is a 2D platformer game featuring interactive objects that present players with various challenges. Players will encounter enemies and navigate through platforms, making for an engaging and dynamic gaming experience.

The main features included in this project have been:

  • Use of multiple Colliders to interact with tile map, the platforms, enemies and items of the game.
  • Simple IA to emulate the enemy movement and attack pattern.
  • Includes collectible items.
  • Power-up and recovery items.
  • Range and close combat actions.
